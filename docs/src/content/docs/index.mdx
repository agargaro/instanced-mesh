## Features

<CardGrid>
  <Card title="Per-instance frustum culling" icon="external">Skip rendering for out-of-view instances.</Card>
  <Card title="Sorting" icon="external">Reduce overdraw and manage transparent objects efficiently.</Card>
  <Card title="Spatial indexing (dynamic BVH)" icon="external">Speed up raycasting and frustum culling.</Card>
  <Card title="Dynamic capacity" icon="external">Add or remove instances seamlessly.</Card>
  <Card title="Per-instance visibility" icon="external">Toggle visibility for each instance individually.</Card>
  <Card title="Per-instance opacity" icon="external">Set opacity for each instance individually.</Card>
  <Card title="Object3D-like instances" icon="external">Use instances like Object3D with transforms and custom data.</Card>
  <Card title="Per-instance uniforms" icon="external">Assign unique shader data to individual instances.</Card>
  <Card title="Level of Detail (LOD)" icon="external">Dynamically adjust instance detail based on distance.</Card>
  <Card title="Shadow LOD" icon="external">Optimize shadow rendering with lower detail for distant instances.</Card>
</CardGrid>

### Install from the package manager 
<Tabs syncKey="pkg">
<TabItem label="npm">

```sh
npm install @three.ez/instanced-mesh
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm add @three.ez/instanced-mesh
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn add @three.ez/instanced-mesh
```

</TabItem>
</Tabs>
